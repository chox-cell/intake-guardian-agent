# Install — Decision Cover™

## Requirements
- Node.js 20+
- pnpm
- macOS or Linux

## Install
pnpm install

## Configure
Create .env.local (DO NOT COMMIT):

ADMIN_KEY=super_secret_admin_123
PORT=7090

## Run
ADMIN_KEY=super_secret_admin_123 pnpm dev

## Open (demo)
http://127.0.0.1:7090/ui/admin?adminKey=super_secret_admin_123
